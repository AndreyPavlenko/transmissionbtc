<p><i>Transmission</i> ist ein leistungsstarker und dennoch extrem leichter und schneller BitTorrent-Client. Es bietet die Funktionen, die Sie von einem BitTorrent-Client erwarten: Verschlüsselung, Webschnittstelle, Peer-Exchange, Magnetlinks, DHT, µTP, UPnP und NAT-PMP-Portweiterleitung, Webseed-Unterstützung, Tracker-Bearbeitung, Geschwindigkeitsbegrenzungen für globale und pro Torrent und mehr.</p><p>Diese App ist ein Port des Transmission-Daemons für Android, der folgende Funktionen enthält:</p><ul><li>Verwalten Sie Downloads direkt aus der Anwendung.</li><li>Mehrere Watch- / Download-Verzeichnisse.</li><li>Unterstützung für HTTP (S) / SOCKS-Proxy.</li><li>WiFi / Ethernet-Modus zum Speichern mobiler Daten.</li><li>Zugelassene WLAN-SSIDs: Wenn konfiguriert, wird der Dienst nur ausgeführt, wenn er mit den angegebenen Netzwerken verbunden ist.</li><li>Behalten Sie die CPU / WLAN-Verbindung bei, um alle Downloads abzuschließen, bevor das Gerät in den Ruhezustand wechselt.</li><li>Sequentieller Download ermöglicht das Abspielen von Mediendateien während des Herunterladens.</li><li>Öffnen Sie Torrent-Dateien oder Torrent- / Magnet-URLs und streamen Sie die ausgewählten Dateien auf einen Media Player.</li><li>Integrierter UPnP MediaServer. Laden Sie Mediendateien auf ein Telefon / Tablet / TV-Box herunter und sehen Sie sich auf einem Fernsehgerät oder einem anderen UPnP-kompatiblen Mediaplayer an, der mit demselben Netzwerk verbunden ist.</li><li>M3U-Wiedergabelisten für alle Torrents / Ordner, die Audio- / Videodateien enthalten. Um die Wiedergabelisten-URL abzurufen, klicken Sie lange auf das Wiedergabesymbol.</li><li>Alternative Weboberfläche - <a href="https://github.com/ronggang/transmission-web-control/wiki" target="_blank" rel="nofollow">Transmission Web Control</a>.</li></ul><p>Der Daemon läuft als Hintergrunddienst. Für die Fernkommunikation mit dem Dämon können Sie entweder die integrierte WEB-Schnittstelle oder eine beliebige Transmission-Fernbedienung verwenden.</p><p>Weitere Informationen finden Sie auf der offiziellen Website des <i>Transmission</i>-Projekts: <a href="https://transmissionbt.com/" target="_blank" rel="nofollow">transmissionbt.com</a>.
